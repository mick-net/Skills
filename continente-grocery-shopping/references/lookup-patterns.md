# Lookup Patterns

Use this file when the task is about product discovery, comparison, or reply formatting.

## Reply format

Reply in English, but include the original Portuguese product name in parentheses.

For each shortlisted option, prefer showing:
- English label
- Portuguese product name in parentheses
- brand
- pack size / units / weight when visible
- price
- unit price when visible

Include a tiny vocab hint when useful.

Example:
- Continente white napkins, 2-ply *(Guardanapos 2 Folhas Branco Continente)* — 100 units — €1.29
- Portuguese: `guardanapos` = napkins; `folhas` = ply/layers

End with a short recommendation when there is an obvious value pick.

## Search guidance

Translate English requests into concise PT-PT search terms.

Examples:
- chicken → `frango`
- beef → `bovino`
- napkins → `guardanapos`
- potatoes → `batata`
- eggs → `ovos`

If the first query is noisy, refine with 1-2 better variants.

## Filtering guidance

Search results can include near-category junk. Keep relevant products, ignore the rest.

Example:
- `guardanapos` may also show kitchen roll or toilet paper. Keep napkins, drop the rest.

## Tested example: napkins

Behavior that worked:
- search `guardanapos`
- filter out unrelated paper products
- present options in English with Portuguese names in parentheses
- include units, price, and value cues
- clarify between 1-ply and 2-ply value-pack options before adding
